# Semantic Mapping

| Dataset Attribute | Meaning | Ontology / Vocabulary Concept |
|---|---|---|
| geo | Country or geographic reporting area | GeoNames / spatial feature |
| subst_cat | Pesticide substance category | Eurostat pesticide substance category |
| TIME_PERIOD | Reporting year | OWL-Time year |
| OBS_VALUE | Observed measured value | Statistical observation value |
| unit | Unit of measurement | SI Digital Framework unit |
| freq | Observation frequency | Statistical frequency |
| OBS_FLAG | Observation status flag | Eurostat observation flag |
| CONF_STATUS | Confidentiality status | Eurostat confidentiality status |

The dataset follows Eurostat statistical data conventions. Domain-specific Eurostat codes are kept for substance categories, countries, flags, and frequency because they are the authoritative controlled terms for this dataset.
