# Unit Mapping

| Attribute | Dataset Unit | Ontology Concept | URI |
|---|---|---|---|
| OBS_VALUE | KG | Kilogram | https://si-digital-framework.org/SI/units/kilogram |
| TIME_PERIOD | Year | Year | http://www.w3.org/2006/time#Year |

The numeric observation value is measured in kilograms. The unit is mapped to the SI Digital Framework concept for kilogram. The time period is represented as a calendar year and mapped to the OWL-Time year concept.
