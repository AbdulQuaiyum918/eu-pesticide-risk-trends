# Eurostat Pesticide Risk Trend Analysis

[![DOI](https://zenodo.org/badge/1231317717.svg)](https://doi.org/10.5281/zenodo.20467709)

## Project Overview

This repository contains the FAIR Data Science project for analysing pesticide risk trends across European Union member states using harmonised Eurostat indicator data.

The project applies FAIR data principles and reproducible workflows for data processing, metadata documentation, machine learning experimentation, evaluation, and long term research data publication.

## Repository Contents

* Eurostat source dataset
* SQL schema and analytical views
* Metadata artefacts
* Machine learning outputs
* Documentation files
* FAIR compliance resources
* Data management plan

## Metadata Standards Implemented

This repository includes implementation of the following FAIR metadata standards:

* RO-Crate
* CodeMeta
* Croissant metadata
* FAIR4ML metadata
* Model Card
* CITATION.cff
* Machine actionable DMP
