# Traffic Accident Severity Austria

[![DOI](https://zenodo.org/badge/1231317717.svg)](https://doi.org/10.5281/zenodo.20467709)

## Project Overview

This repository contains the FAIR Data Science project for analysing traffic accident severity patterns in Austria.

The project applies structured metadata standards and reproducible workflows for data processing, modelling, evaluation, and FAIR publication.

## Repository Contents

- Source dataset
- SQL schema and analytical views
- Metadata artefacts
- Model outputs
- Documentation
- FAIR compliance files

## Metadata Standards Implemented

This project includes:

- RO Crate
- CodeMeta
- Croissant metadata
- FAIR4ML metadata
- Model Card
- Data Management Plan
